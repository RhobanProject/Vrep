Collidable objects

Related API functions

Collidable objects are objects that can be tested for collision against other collidable objects, i.e. that will register a collision state. For now, only shapes are collidable objects.

Collections are also collidable, since they might contain collidable objects.

Collidable objects can have their collidable property individually enabled or disabled (enabled by default for non-pure shapes). This can be set in the object common properties dialog or through the simSetObjectSpecialProperty API function.

Additionally, collidable objects can have their collidable property overridden depending on their related model properties (if they are part of a model). Refer to the model dialog for more information.
